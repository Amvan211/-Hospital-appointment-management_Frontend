# -Hospital-appointment-management
This is my group project (5 members) for developed as part of an Object-Oriented Programming (OOP) course.
# Project Overview
  Develop a fully functional, secure, and user-friendly online appointment scheduling system. The system allows all parties (patients, doctors, and administrators) to interact effectively through a web interface, ensuring a professional and convenient patient care process.
# Tech Stack
  - HTML5, CSS3, JavaScript (ES6+)
  - ReactJS
  - Axios, React Router, Redux / Context API
  - Bootstrap
